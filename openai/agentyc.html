Overview

I'm the solo creator of Agentyc - a macOS app that empowers non-technical users to experiment with AI Agent prompts, LLMs and Skills without writing a single line of code.

Agentyc is powered by Microsoft Autogen for agent orchestration & collaboration, supports LLMs from providers such as Anthropic, OpenAI or Google, and ships with ready-to-use skills for web searching and browsing, giving AI Agents the ability to solve complex tasks that require access to the latest data.

Origin Story

I spent a few months gaining hands-on experience with AI Agent orchestration frameworks such as CrewAI, Microsoft Autogen and OpenAI Swarm. I started diving deeper into Microsoft Autogen because of it's flexibility in spawning Agents and their ability to self-correct mistakes made during task execution.

Microsoft had released Autogen Studio: a GUI for rapid experimentation with Agents, Models and Prompts. However, I found this UI was not building the right mental model for new users and there were other areas of improvement.

I took on the challenge of creating an alternative to Autogen Studio to empower non-technical users to experiment with Agents and build the right mental model of an AI Agent ecosystem.

Product Goals

Drastically simplify the installation & onboarding process specifically for non-technical users
Craft workflows to help users develop the right mental model of an AI Agent ecosystem
Provide basic, ready-to-use Skills for web searching and browsing using open source libraries
Support interoperability with Autogen Studio to take advantage of its community features

The Building Blocks

Agents

Agents are entities that are assigned a role and specialization by the user in plain English. For example, a research agent will specialize in searching the web and browsing web pages to gather the latest data. An analyst Agent can take the data gathered by the research Agent and derive insights from it.

Models

Models are the brains behind Agents. Each Agent in a team can be assigned a different model to cater to their specialization. For example, a coding Agent can be linked to Claude 3.5 Sonnet and a journalist Agent can be linked to OpenAI GPT 4.0.

Skills

Skills are Python functions that enable Agents to search the web, browse webpages or retrieve data from third-party APIs. Without Skills, Agents only have access to the data trained in the Model and therefore have limited capabilities.

Missions

Missions are tasks with well-defined outputs that are assigned to a group of Agents. Each Mission can be deployed over multiple Sessions with changes to prompts, models and skills to see variations in output.
