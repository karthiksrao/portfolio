Competitor Analysis

To identify opportunities to improve Character AI's user experience, I conducted a UX review of the top 3 competitors.

ChAI

👍 Supports chatting with one AI character & humans, but not like "rooms"
✨ Allows users to swap the LLM behind each character
🤯 Developers can deploy custom models & earn revenue
👎 Personal connection to a character feels very weak
👎 Models may not be fine-tuned to match their character
👎 Generic UI design

Replika

🤯 Custom generated 3D model for every user-created character
👍 Freemium model combined with paid consumables works really well
🔥 Personal connection to character is the strongest among all 3
👎 Supports chatting with only a single character
👎 3D models look cartoonish, hard to take seriously
👎 Can feel like a videogame at times

Talkie

✨ Leverages GenAI to create avatars for characters from a reference image
👍 Builds personal connection with a character by tracking "relationship levels"
👍 Unique visual design which is consistent with the fantasy theme
🤷‍♂️ Skewed towards fantasy & sci-fi, may not appeal to a mainstream audience
👎 Some anime/fantasy jargon used in the UI, which can alienate mainstream users
👎 UI can be overwhelming

Design Goals

By comparing Character AI's current experience to the competition, I identified the following design goals I would set for myself.

  • Facilitate users in building a personal connection to characters, which will increase user retention and new user adoption.
  • Discover more opportunities for monetization or added benefits for C.AI+ users, including brand & celebrity partnerships.
  • Unlock ways to make Character AI an irreplacable app by providing more than entertainment value.
  • Design a seamless and consistent experience across all platforms to support higher modalities,
    including niche ones like Apple Vision Pro.
