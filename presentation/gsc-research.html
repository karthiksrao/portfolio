User Needs & Goals

User Groups

I identified 3 user groups and conducted interviews to understand their needs.

Domain Admin

User Needs

  • Self-sufficient way to manage my channel.
  • Tailor answers to colleague's region.
  • Track incoming user queries and question variations.

Service Director

User Needs

  • Visibility into my channel's performance metrics.
  • Human agent management.
  • Domain-level user and permission management.

Genie Admin

User Needs

  • Create and manage top-level domains such as Technology, People etc.
  • Visibility into all channels.
  • App-level user and permission management
